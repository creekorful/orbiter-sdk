Orbiter Scenario Editor

Save a scenario

Once you are happy with the scenario you have created, you can save it to a scenario file which you can run at a later time or share with other orbiter users.

To save the current simulation state from within the scenario editor, click the Save button on the main editor page. This will open the Save scenario page.

Enter a file name and a short scenario description. Optionally, the file name can include a directory path. All paths are relative to Orbiter's root scenario folder (Orbiter\Scenarios). You can enter a sub-folder in the file name box (e.g. MyScenarios\Interplanetary\MarsTransfer) but the sub-folder must already exist.

The text you enter in the description box will be displayed in Orbiter's Launchpad dialog when a user selects the scenario from the list. It should briefly explain what the scenario is about.

Finally, click Save to save the scenario to file, or Cancel to return without saving.
